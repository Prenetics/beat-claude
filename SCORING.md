# Scoring Rubric

Every submission is blind-reviewed against Claude's baseline answer on the same challenge. Scores are out of 100.

## Dimensions

### Strategic Depth (30 points)

| Score | Description |
|---|---|
| 25–30 | Identifies non-obvious constraints, considers second-order effects, reasoning is airtight |
| 18–24 | Sound strategy with clear logic, addresses key constraints |
| 10–17 | Surface-level strategy, misses important considerations |
| 0–9 | No clear strategic thinking |

**We ask:** Does this person see around corners? Did they catch something Claude missed?

### Specificity (25 points)

| Score | Description |
|---|---|
| 21–25 | Concrete numbers, named tools/platforms, defined timelines, immediately executable |
| 14–20 | Mostly specific with some actionable detail |
| 7–13 | Vague recommendations, generic advice |
| 0–6 | No actionable content |

**We ask:** Could someone take this plan and execute it tomorrow morning?

### AI Fluency (20 points)

| Score | Description |
|---|---|
| 17–20 | Demonstrates sophisticated understanding of AI capabilities and limitations, proposes AI-enhanced workflows |
| 11–16 | Shows practical AI knowledge, identifies automation opportunities |
| 5–10 | Mentions AI but superficially |
| 0–4 | No AI awareness |

**We ask:** Does this person know how to make AI work for them, not just parrot it?

### Creativity (15 points)

| Score | Description |
|---|---|
| 13–15 | Unique angles, cross-industry pattern recognition, contrarian-but-sound ideas |
| 9–12 | Some original thinking mixed with conventional approaches |
| 4–8 | Entirely conventional playbook |
| 0–3 | No original thought |

**We ask:** Did they surprise us? Would Claude ever come up with this?

### Communication (10 points)

| Score | Description |
|---|---|
| 9–10 | Crystal clear, well-structured, appropriate depth for the audience |
| 6–8 | Clear and organized with minor issues |
| 3–5 | Disorganized or unclear in places |
| 0–2 | Difficult to follow |

**We ask:** Would we put this in front of our CEO?

## Outcome

- **Beat Claude's score** → Interview invitation within 5 business days
- **Within 5 points of Claude** → Tiebreaker (follow-up questions or brief call)
- **Below Claude's score** → We keep your application on file

## Blind Review Process

1. Submissions are anonymized — reviewer sees "Candidate A" vs "Candidate B" (one is Claude)
2. Both answers are scored independently on the rubric
3. Scores are compared only after both are graded
4. No bias, no shortcuts
